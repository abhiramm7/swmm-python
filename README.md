# swmm-python

Home to Python packages for OWA SWMM.


## Contents:
  nrtest-swmm - Plugin for performing numerical regression testing on OWA swmm-solver.

  swmm-toolkit - SWIG based wrappers for the OWA swmm-solver and swmm-output libraries.
